Basic Terms

1) Computer - A computer can be defined as a programmable, electronic device that accepts data, per forms operations on that data, presents the results, and stores the data or results as needed.

2) Processing - Performing operations on data that has been input into a computer to convert that input to output.

3) Communication - The transmission of data from one device to another.

4) Storage - The operation of saving data, programs or output for future use.

5) Hardware - The physical parts of a computer system, such as the keyboard, monitor, printer, and so forth.

6) Software - The instructions, also called computer programs, that used to tell a computer what it should do.

7) Operating system â€“ The main component of system software that enables a computer to operate, manage its activities and the resources under its control , run application programs, and interface with the user.

8) Application software - Programs that enable users to perform specific tasks on a computer, such as writing letters or playing; games; also called apps.

9) Server - A computer used to host programs and data for a small network.

10) Mainframe computer - A computer used in large organizations(such as hospitals, large businesses and colleges) that need to manage large amounts of centralized data and run multiple programs simultaneously.

11) Supercomputers - The fastest, most expensive and most powerful type of computer.

12) Computer network - A collection of computers and other hardware devices that are connected together to share hardware, software and data as well as to communicate electronically with one another.

13) Internet - The largest and most well-known computer network, liking millions of computers all over the world.

14) Web page - A document, typically containing hyperlinks to other documents, located on a web server and available through the world wide web. (Morley 26)

15) Web site - A collection of related web pages usually belonging to an organization or individual.

16) System Unit - Case that contains the CPU, memory power supply, storage, device and all other internal hardware.

17) Information - Data that has been processed into a meaningful form. (Morley 9)

18) Embedded computer - A tiny computer embedded in a product and designed to perform specific tasks or functions for that product.

19) Mobile device - A very small device that has built-in computing or internet capability.

20) Internet address - An address that identifies a computer. Person or webpage on the internet, such as an IP address, domain name, or email address.
